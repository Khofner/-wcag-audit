FAQ

Everything you need to know about the WCAG Audit dashboard

Getting started
An AI-assisted WCAG 2.2 accessibility audit dashboard for development and design teams. It combines AI screenshot analysis, code review with auto-fix, and team collaboration in one place.
Visit console.anthropic.com/settings/keys, sign in, click "Create key", and copy it. It starts with sk-ant-. Paste it into the API Key field in the sidebar.

Tools
All 54 WCAG 2.2 criteria, categorized as AI-reviewed, Partial, or Manual. AI-reviewed criteria include alt text, contrast, heading structure, label presence, and target sizes.
Paste code, run review, apply HTML fixes directly in the editor, click Run Again to re-analyze. Typically 2–3 passes resolve all auto-fixable issues. CSS/JS/Design findings are tagged for your team.
Roadmap
Persistent audit history via Supabase, team login with magic link auth, compliance trend charts, and Jira ticket creation from findings. When a ticket closes in Jira, the finding auto-resolves.
